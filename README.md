## 上海交大×商汤科技【OpenMMLab实践公开课】

上海交大×商汤科技【OpenMMLab实践公开课】由上海交大学生创新中心与商汤科技联合导师团共同授课，课程涵盖计算机视觉中的分类、检测、分割等经典领域与图像生成GAN、字符识别等有趣案例，从经典问题入手深入浅出讲解理论知识并结合OpenMMLab代码库进行实战，体验案例分析、数据标注、代码复现、模型部署的全流程
详情见[上海交大×商汤科技【OpenMMLab实践公开课】](./lecture_sjtu.md)
